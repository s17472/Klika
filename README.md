# Problem kliki
## Dane: 
* K - tablica z listą sąsiedztwa dla wierzchołów grafu 
* P - tablica wierzchołków z częściowym wynikiem

## Modele: 
### Wierzchołek: 
* Id - id 
* X - współrzędna x
* Y - współrzędna Y
### Tablica K
* Id_W - id wierzchołka
* Id_S - lista id sąsiedzkich wierzchołków 
### Tablica P
- Id_W - id wierzchołka
